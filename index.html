Olá! Eu sou Gabriel Resende e recentemente decidi trilhar o caminho da programação. No momento, estou cursando Análise e Desenvolvimento de Sistemas (ADS) e me dedicando por meio de cursos na área da tecnologia para aprimorar minhas habilidades.

Habilidades

HTML, CSS e JavaScript

HTML, CSS e JavaScript são as principais tecnologias para criar aplicativos web. HTML organiza o conteúdo, CSS controla o estilo e JavaScript adiciona interatividade. Dominar essas linguagens é essencial para desenvolver experiências web atrativas e dinâmicas.

Bootstrap

O Bootstrap é um framework de código aberto para desenvolvimento web que fornece componentes pré-projetados e estilos CSS. Ele facilita a criação de sites e aplicativos responsivos, economizando tempo e esforço ao oferecer elementos de design prontos para uso.

Git e Github

Git é um sistema de controle de versão que rastreia mudanças em projetos de software, permitindo colaboração e gerenciamento eficiente. GitHub é uma plataforma online que utiliza o Git para hospedar, compartilhar e colaborar em projetos, facilitando revisões de código, rastreamento de problemas e contribuições abertas.
